# IBM Internship CRUD Application

This project is a simple full-stack CRUD application developed as part of the IBM Internship assignment.

## Technologies Used
- Node.js
- Express.js
- HTML
- JavaScript

## Project Structure
- backend: Server-side logic using Express
- frontend: Client-side HTML and JavaScript

## Features
- Add student details
- View student list
- Delete student records

## How to Run the Project
1. Install dependencies using:
   npm install
2. Start the backend server:
   node backend/server.js
3. Open frontend/index.html in browser

## Output
The application allows users to perform basic CRUD operations on student data.
